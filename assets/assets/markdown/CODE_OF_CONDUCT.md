# Code of conduct
I expect from the contributors to act professionally and respectfully.

Specifically:

* Respect Kashmiri Literature.
* Respect the work of contributors.
* Be kind. Be courteous. Be welcoming.

## Conflict resolution

When multiple contributors disagree on the direction for a particular
patch or the general direction of a particular word, the conflict will be resolved by language professionals. 

## Questions

Feel free to contact me if you have any doubts regarding app.
