# Kaeshir Dictionary

An offline two way Kashmiri-English dictionary.
# Description
It is one of the simplest Kashmiri dictionary available which is still in growing phase.
Unlike other apps, this app has the advantage that many of its features work offline. Material design is used for elegant UI and is topped by a search bar where you enter words you want to look up with a sliding drawer.

Intelligent search bar is there which filters both Kashmiri as well as English words. You can also suggest new words which may be included in the dictionary if the new words are found authentic.
